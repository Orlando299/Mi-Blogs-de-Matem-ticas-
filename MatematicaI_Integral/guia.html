1. Integral indefinida y métodos de integración

Definición 1.1. \(\int f(x)dx = F(x)+C\) donde \(F'(x)=f(x)\).
Integrales inmediatas: \(\int x^n dx = \frac{x^{n+1}}{n+1}+C\ (n\neq-1)\), \(\int \frac{1}{x}dx=\ln|x|+C\), \(\int e^x dx=e^x+C\), \(\int \sin x\,dx=-\cos x+C\).
Métodos: sustitución, integración por partes (\(\int u\,dv = uv - \int v\,du\)), fracciones parciales.
Ejercicio 1.1. \(\int (3x^2-4x+2)dx\).
\(x^3-2x^2+2x+C\).
Ejercicio 1.2. \(\int x e^{x^2}dx\).
\(\frac{1}{2}e^{x^2}+C\).
Ejercicio 1.3. \(\int \ln x\,dx\).
\(x\ln x - x + C\).

2. Integral definida y Teorema Fundamental

Definición 2.1. \(\int_a^b f(x)dx = \lim_{n\to\infty}\sum f(x_i^*)\Delta x\).
Teorema Fundamental del Cálculo: Si \(F'=f\), entonces \(\int_a^b f(x)dx = F(b)-F(a)\).
Ejercicio 2.1. \(\int_0^2 (x^3-2x)dx\).
0.
Ejercicio 2.2 (Valor eficaz). Para \(i(t)=I_m\sin(\omega t)\), calcula \(I_{rms}=\sqrt{\frac{1}{T}\int_0^T i^2(t)dt}\).
\(I_{rms}=I_m/\sqrt{2}\).

3. Aplicaciones de la integral

Área entre curvas: \(A=\int_a^b |f(x)-g(x)|dx\).
Volumen de revolución (disco): \(V=\pi\int_a^b [R(x)]^2 dx\).
Trabajo y energía: \(W=\int F(x)dx\), \(E=\int P(t)dt\).
Ejercicio 3.1. Área entre \(y=x^2\) y \(y=2x-x^2\).
\(\int_0^1 (2x-2x^2)dx = \frac{1}{3}\).
Ejercicio 3.2 (Energía). La potencia en una resistencia es \(P(t)=V_m I_m \sin^2(\omega t)\). Calcula la energía disipada en un periodo.
\(E = V_{rms}I_{rms} T\).

4. Series infinitas

Serie numérica: \(\sum_{n=1}^\infty a_n\). Converge si las sumas parciales tienen límite finito.
Serie de Taylor: \(f(x)=\sum_{n=0}^\infty \frac{f^{(n)}(a)}{n!}(x-a)^n\).
Ejercicio 4.1. Determina la convergencia de \(\sum_{n=1}^\infty \frac{1}{n^2}\).
Converge (serie p con \(p=2>1\), suma \(\pi^2/6\)).
Ejercicio 4.2 (Aproximación). Usa la serie de Maclaurin de \(e^x\) para aproximar \(e^{0.5}\) con 3 términos.
\(1 + 0.5 + 0.125 = 1.625\).
