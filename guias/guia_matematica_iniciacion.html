📌 Unidad 1: Factorización, Potenciación y Racionalización

Factorización: Escribir una expresión algebraica como producto de factores irreducibles.
Potenciación: \(a^n = a \cdot a \cdots a\) (n veces). Propiedades: \(a^m a^n = a^{m+n}\), \((a^m)^n = a^{mn}\), \(a^{-n}=1/a^n\).
Racionalización: Eliminar radicales del denominador.

Casos de factorización importantes

  • Diferencia de cuadrados: \(a^2 - b^2 = (a+b)(a-b)\)
  • Trinomio cuadrado perfecto: \(a^2 \pm 2ab + b^2 = (a \pm b)^2\)
  • Suma/diferencia de cubos: \(a^3 \pm b^3 = (a \pm b)(a^2 \mp ab + b^2)\)
⚡ Aplicación eléctrica: Al calcular impedancias en paralelo: \(Z = \frac{R \cdot j\omega L}{R + j\omega L}\). Para separar parte real e imaginaria, racionalizamos multiplicando por el conjugado.
Ejercicio 1.1 – Factoriza: \(x^2 - 16\)
Solución: \((x-4)(x+4)\)
Ejercicio 1.2 – Racionaliza: \(\frac{5}{\sqrt{3}}\)
Solución: \(\frac{5\sqrt{3}}{3}\)
Ejercicio 1.3 (Eléctrico) – Si \(Z = 4+3j\) y \(V=10\), calcula la corriente \(I = V/Z\) racionalizando.
Solución: \(I = \frac{10}{4+3j} = \frac{10(4-3j)}{25} = \frac{40-30j}{25} = 1.6 - 1.2j\) A

📌 Unidad 2: Sistema de Coordenadas Cartesianas y Funciones Reales

Plano cartesiano: Par ordenado \((x,y)\). Distancia entre dos puntos: \(d = \sqrt{(x_2-x_1)^2+(y_2-y_1)^2}\).
Función real: Relación que asigna a cada \(x\) un único \(y = f(x)\). Dominio: valores de \(x\) con imagen real.
Ejercicio 2.1 – Halla el dominio de \(f(x)=\frac{x+1}{x^2-4}\).
Solución: \(x^2-4 \neq 0 \Rightarrow x \neq \pm2\). Dominio \(\mathbb{R}\setminus\{-2,2\}\).
Ejercicio 2.2 – Distancia entre \(A(2,-1)\) y \(B(5,3)\).
Solución: \(\sqrt{(5-2)^2+(3+1)^2} = \sqrt{9+16}=5\).

📌 Unidad 3: Funciones Trigonométricas y Logarítmicas

Identidades: \(\sin^2\theta + \cos^2\theta = 1\), \(\tan\theta = \frac{\sin\theta}{\cos\theta}\).
Logaritmos: \(\ln(ab)=\ln a + \ln b\), \(\ln(a^n)=n\ln a\), \(\log_a x = \frac{\ln x}{\ln a}\).
Ejercicio 3.1 – Resuelve \(\log_2(x) + \log_2(x-2)=3\).
Solución: \(\log_2(x(x-2))=3 \Rightarrow x(x-2)=8 \Rightarrow x^2-2x-8=0 \Rightarrow x=4\) (descartamos \(x=-2\)).
Ejercicio 3.2 – Simplifica \(\frac{\sin^2 x}{1-\cos x}\).
Solución: \(\frac{1-\cos^2 x}{1-\cos x} = 1+\cos x\).

📌 Unidad 4: Determinantes y Matrices

Matriz \(A_{m\times n}\). Determinante de 2x2: \(\det\begin{pmatrix}a&b\\c&d\end{pmatrix}=ad-bc\).
Para 3x3: Regla de Sarrus.
Ejercicio 4.1 – Calcula \(\det\begin{pmatrix}2&5\\-1&3\end{pmatrix}\).
Solución: \(2\cdot3 - 5\cdot(-1)=6+5=11\).
Ejercicio 4.2 – Halla el determinante de \(A=\begin{pmatrix}1&2&3\\0&1&4\\5&6&0\end{pmatrix}\).
Solución: \(1(1\cdot0-4\cdot6)-2(0\cdot0-4\cdot5)+3(0\cdot6-1\cdot5)= -24+40-15=1\).

📌 Unidad 5: Sistemas de Ecuaciones Lineales

Métodos: sustitución, igualación, reducción (Gauss), regla de Cramer.
Aplicación en circuitos: leyes de Kirchhoff generan sistemas de ecuaciones.
Ejercicio 5.1 – Resuelve \(\begin{cases}3x+4y=18\\2x-y=1\end{cases}\).
Solución: \(x=2, y=3\).
Ejercicio 5.2 (Eléctrico) – Dado el sistema de corrientes: \(I_1+I_2=I_3\), \(2I_1+4I_3=12\), \(3I_2+4I_3=10\). Halla \(I_1,I_2,I_3\).
Solución: Resolviendo se obtiene \(I_1=22/13 \approx 1.6923\) A, \(I_2=6/13 \approx 0.4615\) A, \(I_3=28/13 \approx 2.1538\) A.

📌 Unidad 6: Nociones Básicas de Números Complejos

Forma binómica: \(z = a + bj\) (en ingeniería se usa \(j\) en lugar de \(i\)).
Módulo: \(|z| = \sqrt{a^2+b^2}\). Argumento: \(\theta = \arctan(b/a)\).
Forma polar: \(z = |z|(\cos\theta + j\sin\theta) = |z| e^{j\theta}\).
Impedancia \(Z = R + jX\). Por ejemplo, \(Z=10+10j\) tiene módulo \(10\sqrt{2}\) y fase \(45^\circ\).
Ejercicio 6.1 – Dados \(z_1=3+4j\), \(z_2=1-2j\), calcula \(z_1/z_2\).
Solución: \(\frac{3+4j}{1-2j} \cdot \frac{1+2j}{1+2j} = \frac{-5+10j}{5} = -1+2j\).
Ejercicio 6.2 – Convierte \(z = -2+2j\sqrt{3}\) a forma polar.
Solución: Módulo \(4\), argumento \(120^\circ\) (2π/3 rad). Forma polar: \(4\angle 120^\circ\).

⚡ Problemas integradores (aplicación a electricidad)

Problema 1 – Una tensión \(v(t)=100\sin(314t)\) se aplica a una carga \(Z=20+15j\ \Omega\). Expresa la corriente fasorial y en el tiempo.
Solución: \(|Z|=25,\ \phi=36.87^\circ\). \(I = \frac{100\angle0^\circ}{25\angle36.87^\circ}=4\angle-36.87^\circ\) A. En el tiempo: \(i(t)=4\sin(314t-36.87^\circ)\) A.
Problema 2 – Un transformador ideal tiene relación \(N_p/N_s=8\). Si \(V_p=1200\) V, halla \(V_s\). Si \(I_s=10\) A, halla \(I_p\).
Solución: \(V_s=150\) V, \(I_p=1.25\) A.
