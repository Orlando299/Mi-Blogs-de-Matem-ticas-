1. Factorización, Potenciación y Racionalización

Definición 1.1 (Factorización). Escribir una expresión algebraica como producto de factores irreducibles.
Definición 1.2 (Potenciación). \(a^n = \underbrace{a\cdot a\cdots a}_{n\text{ veces}}\).
Definición 1.3 (Racionalización). Eliminar radicales del denominador.
Ejemplo 1.1 (Aplicación eléctrica). Al calcular impedancias en paralelo: \(Z = \frac{R \cdot j\omega L}{R + j\omega L}\). Racionalizando se obtiene parte real e imaginaria.
Ejercicio 1.1. Factoriza: \(x^2 - 16\).
Solución: \((x-4)(x+4)\).
Ejercicio 1.2. Racionaliza: \(\frac{5}{\sqrt{3}}\).
Solución: \(\frac{5\sqrt{3}}{3}\).
Ejercicio 1.3 (Eléctrico). Si \(Z = 4+3j\) y \(V = 10\), calcula \(I = V/Z\) racionalizando.
Solución: \(I = \frac{10}{4+3j} = \frac{10(4-3j)}{25} = 1.6 - 1.2j\) A.

2. Sistema de Coordenadas Cartesianas y Funciones Reales

Definición 2.1. El plano cartesiano asigna a cada punto un par \((x,y)\). Distancia: \(d = \sqrt{(x_2-x_1)^2+(y_2-y_1)^2}\).
Ejercicio 2.1. Halla el dominio de \(f(x)=\frac{x+1}{x^2-4}\).
Solución: \(x \neq \pm2\), dominio \(\mathbb{R}\setminus\{-2,2\}\).
Ejercicio 2.2. Distancia entre \(A(2,-1)\) y \(B(5,3)\).
Solución: \(\sqrt{9+16}=5\).

3. Funciones Trigonométricas y Logarítmicas

Identidades: \(\sin^2\theta+\cos^2\theta=1\), \(\tan\theta=\frac{\sin\theta}{\cos\theta}\).
Logaritmos: \(\ln(ab)=\ln a+\ln b\), \(\ln(a^n)=n\ln a\).
Ejercicio 3.1. Resuelve \(\log_2(x)+\log_2(x-2)=3\).
Solución: \(x=4\) (descartamos \(x=-2\)).
Ejercicio 3.2. Simplifica \(\frac{\sin^2 x}{1-\cos x}\).
Solución: \(1+\cos x\).

4. Determinantes y Matrices

Determinante 2x2: \(\det\begin{pmatrix}a&b\\c&d\end{pmatrix}=ad-bc\).
Ejercicio 4.1. Calcula \(\det\begin{pmatrix}2&5\\-1&3\end{pmatrix}\).
Solución: \(2\cdot3 - 5\cdot(-1)=11\).
Ejercicio 4.2. Halla \(\det\begin{pmatrix}1&2&3\\0&1&4\\5&6&0\end{pmatrix}\).
Solución: \(1(1\cdot0-4\cdot6)-2(0\cdot0-4\cdot5)+3(0\cdot6-1\cdot5)= -24+40-15=1\).

5. Sistemas de Ecuaciones Lineales

Ejemplo 5.1 (Circuitos). Las leyes de Kirchhoff generan sistemas.
Ejercicio 5.1. Resuelve \(\begin{cases}3x+4y=18\\2x-y=1\end{cases}\).
Solución: \(x=2, y=3\).
Ejercicio 5.2 (Eléctrico). \(I_1+I_2=I_3\), \(2I_1+4I_3=12\), \(3I_2+4I_3=10\). Halla las corrientes.
Solución: \(I_1\approx1.6923\) A, \(I_2\approx0.4615\) A, \(I_3\approx2.1538\) A.

6. Nociones Básicas de Números Complejos

Forma binómica: \(z = a + bj\) (en ingeniería se usa \(j\)).
Módulo: \(|z| = \sqrt{a^2+b^2}\). Forma polar: \(z = |z| e^{j\theta}\).
Ejemplo 6.1. Impedancia \(Z=10+10j\) tiene módulo \(10\sqrt{2}\) y fase \(45^\circ\).
Ejercicio 6.1. Dados \(z_1=3+4j\), \(z_2=1-2j\), calcula \(z_1/z_2\).
Solución: \(-1+2j\).
Ejercicio 6.2. Convierte \(z = -2+2j\sqrt{3}\) a forma polar.
Solución: \(4\angle 120^\circ\).

Problemas integradores

Problema 1. Una tensión \(v(t)=100\sin(314t)\) se aplica a una carga \(Z=20+15j\ \Omega\). Halla la corriente fasorial y en el tiempo.
Solución: \(I = 4\angle -36.87^\circ\) A, \(i(t)=4\sin(314t-36.87^\circ)\) A.
Problema 2. Transformador ideal: \(N_p/N_s=8\), \(V_p=1200\) V, \(I_s=10\) A. Halla \(V_s\) e \(I_p\).
Solución: \(V_s=150\) V, \(I_p=1.25\) A.
