1. Ecuaciones lineales con una variable

Una ecuación lineal con una variable \(x\) es de la forma: \[ a x + b = 0 \quad (a \neq 0) \] Su solución es única: \(x = -\frac{b}{a}\).
Ejemplo 1: \(3x - 5 = 0\) → \(3x = 5\) → \(x = \frac{5}{3}\).
Ejemplo 2: \(\frac{2x}{3} + 4 = 10\) → \(\frac{2x}{3} = 6\) → \(2x = 18\) → \(x = 9\).

2. Ecuaciones lineales con dos variables

Una ecuación lineal con dos variables \(x\) e \(y\) se escribe como: \[ a x + b y = c \] donde \(a\) y \(b\) no son ambos cero. Tiene infinitas soluciones \((x,y)\) que geométricamente representan una recta en el plano cartesiano.
Ejemplo: \(2x - 3y = 6\). Algunas soluciones: \((3,0)\), \((0,-2)\), \((6,2)\).

3. Sistemas de dos ecuaciones lineales con dos incógnitas (2×2)

Un sistema de dos ecuaciones lineales con dos incógnitas tiene la forma: \[ \begin{cases} a_1 x + b_1 y = c_1 \\ a_2 x + b_2 y = c_2 \end{cases} \] Resolverlo significa encontrar todos los pares \((x,y)\) que satisfacen ambas ecuaciones simultáneamente.

Método de sustitución

Pasos:
  1. Despejar una variable de una ecuación.
  2. Sustituir esa expresión en la otra ecuación.
  3. Resolver la ecuación resultante (una variable).
  4. Sustituir el valor obtenido para hallar la otra variable.
\(\begin{cases} x + 2y = 8 \\ 3x - y = 3 \end{cases}\)
De la primera: \(x = 8 - 2y\). Sustituyendo en la segunda: \(3(8-2y) - y = 3 \rightarrow 24 - 6y - y = 3 \rightarrow 24 - 7y = 3 \rightarrow -7y = -21 \rightarrow y = 3\). Luego \(x = 8 - 2(3) = 2\). Solución: \((2,3)\).

Método de igualación

Pasos:
  1. Despejar la misma variable de ambas ecuaciones.
  2. Igualar las expresiones obtenidas.
  3. Resolver la ecuación resultante.
  4. Sustituir para hallar la otra variable.
\(\begin{cases} 2x - y = 5 \\ x + y = 4 \end{cases}\)
Despejamos \(y\): \(y = 2x - 5\) y \(y = 4 - x\). Igualamos: \(2x - 5 = 4 - x \rightarrow 3x = 9 \rightarrow x = 3\). Luego \(y = 4 - 3 = 1\). Solución: \((3,1)\).

Método de reducción (suma/resta)

Pasos:
  1. Multiplicar una o ambas ecuaciones para que los coeficientes de una variable sean opuestos.
  2. Sumar las ecuaciones para eliminar esa variable.
  3. Resolver la ecuación resultante.
  4. Sustituir para hallar la otra variable.
\(\begin{cases} 3x + 4y = 10 \\ 2x - 4y = 0 \end{cases}\)
Sumamos: \(5x = 10 \rightarrow x = 2\). Sustituyendo en la segunda: \(2(2) - 4y = 0 \rightarrow 4 = 4y \rightarrow y = 1\). Solución: \((2,1)\).

4. Sistemas de tres ecuaciones lineales con tres incógnitas (3×3)

Forma general: \[ \begin{cases} a_1 x + b_1 y + c_1 z = d_1 \\ a_2 x + b_2 y + c_2 z = d_2 \\ a_3 x + b_3 y + c_3 z = d_3 \end{cases} \]

Método de eliminación (Gauss)

Se combinan ecuaciones para eliminar una variable, reduciendo el sistema a uno 2×2, luego se resuelve y se sustituye hacia atrás.
\(\begin{cases} x + y + z = 6 \\ 2x - y + z = 3 \\ x - y + 2z = 5 \end{cases}\)
Paso 1: Eliminamos \(y\) de las dos primeras: sumamos 1ª y 2ª → \(3x + 2z = 9\).
Paso 2: Eliminamos \(y\) de 1ª y 3ª: restamos 1ª menos 3ª → \(2y - z = 1\).
(Ahora tenemos un sistema 2×2 en \(x,z\) y una ecuación con \(y,z\). Se continúa resolviendo.)
Solución final: \((x,y,z) = (1,2,3)\).

5. Aplicaciones en electricidad (Leyes de Kirchhoff)

En un circuito de dos mallas, las corrientes cumplen: \[ \begin{cases} R_1 I_1 + R_2 (I_1 - I_2) = V_1 \\ R_3 I_2 + R_2 (I_2 - I_1) = -V_2 \end{cases} \] Este es un sistema lineal 2×2 que se resuelve con cualquiera de los métodos anteriores.
