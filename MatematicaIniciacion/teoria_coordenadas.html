Sistema de Coordenadas Cartesianas

MSc. Orlando Silva

1. Plano cartesiano y ubicación de puntos

El plano cartesiano está formado por dos ejes perpendiculares: \(x\) (horizontal) e \(y\) (vertical). Cada punto se representa como \((x,y)\).

2. Distancia entre dos puntos

Dados \(P_1(x_1,y_1)\) y \(P_2(x_2,y_2)\), la distancia es: \[ d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2} \]

3. Pendiente de una recta

La pendiente \(m\) de la recta que pasa por dos puntos es: \[ m = \frac{y_2-y_1}{x_2-x_1} \]

4. Ecuación de la recta

  • Forma punto‑pendiente: \(y - y_1 = m(x - x_1)\)
  • Forma pendiente‑ordenada: \(y = mx + b\)
  • Forma general: \(Ax + By + C = 0\)

5. Clasificación de triángulos mediante Pitágoras

Para un triángulo de lados \(a,b,c\) (con \(c\) el mayor):
  • Rectángulo si \(c^2 = a^2 + b^2\)
  • Acutángulo si \(c^2 < a^2 + b^2\)
  • Obtusángulo si \(c^2 > a^2 + b^2\)

6. Intersección de dos rectas

Se resuelve el sistema formado por sus ecuaciones. Gráficamente es el punto donde se cruzan.
